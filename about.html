About

NoLimit was developed by Stevie Hetelekides (hetelek) using the jQuery Mobile user interface system, as well as some original code from the Snaphax class.

NoLimit allows you to view Snapchats for as long as you want, without notifying the sender. Saving is also possible by tapping and holding an image. After opening Snapchats (from the Snapchat application), you will no longer be able to view them through NoLimit.

NoLimit is not affiliated with Snapchat, Inc.
